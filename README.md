# Tanks!

This is my first video game built with Unity3D.

## Getting Started

```
git clone git@github.com:sanlouise/tanks_unity3d.git

cd tanks_unity3d
```

[Download](https://unity3d.com) and open Unity3D (select the tanks_unity3d folder you just cloned).
